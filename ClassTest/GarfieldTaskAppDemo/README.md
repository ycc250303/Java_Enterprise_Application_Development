## 项目结构

* `GarfieldTaskApp`: 项目主程序
* `GarfieldTaskCommand`: 项目命令控制
* `commands\`: 命令示例
* `repository\`: 任务管理仓库
* `service\`: 番茄钟、json生成服务
* `task\`: 任务信息

## 项目运行

* 开启IEDA，在`GarfieldTaskApp.java`文件中用`ctrl+F5`启动，或者点击图形化启动按钮
